---
description: Prepare a draft for manual publishing handoff
---

Prepare `$ARGUMENTS` for publishing.

Requirements:

- Read the draft and check that metadata is complete.
- Produce a concise publishing handoff with:
  - title
  - meta title
  - meta description
  - slug
  - suggested category
  - suggested tags
  - featured image brief
  - internal links to verify
  - external links to verify
- Save the handoff to `published/<draft-slug>-handoff-<YYYY-MM-DD>.md`.

This command should not assume WordPress or any specific CMS unless the draft or site context says so.
